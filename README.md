# Flexa WaitFree Next.js

### 주요 기술 스택:

- Next.js 15
- React 19
- Tailwind CSS 3
- shadcn/ui
- plotly.js
- Supabase

### 실행 명령어:

package.json의 scripts 참조

```shell
# 예시
$ npm install
$ npm run dev
```
